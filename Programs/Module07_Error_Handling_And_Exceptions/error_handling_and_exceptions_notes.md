**Error Handling & Exception**

**Block in Error Handling:**
- try (Mandatory)
- except (Mandatory)
- else (Optional)
- finally (Optional)

---

**Type of Exception Classes:**
- SyntaxError
- TypeError
- ValueError
- KeyError
- ZeroDivisionError
- IOError
- IndexError
- NameError
- AttributeError
- ImportError

---

**Traceback:**
- absolute path of the file & error line number
- prints error'ed line & it denotes the error'ed area (^)
