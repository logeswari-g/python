**Questions on Loops in Python**

## **1. Print Numbers from 1 to 10**
Write a Python program using a `for` loop to print numbers from 1 to 10.

## **2. Sum of First N Natural Numbers**
Write a Python program that takes an integer `N` as input and uses a `while` loop to calculate the sum of the first `N` natural numbers.

## **3. Multiplication Table**
Write a Python program that takes an integer as input and prints its multiplication table using a `for` loop.

## **4. Count Even and Odd Numbers in a List**
Given a list of numbers, use a loop to count how many numbers are even and how many are odd.

## **5. Reverse a String**
Write a Python program that takes a string as input and prints it in reverse using a loop.

## **6. Find Factorial of a Number**
Write a Python program that takes an integer input and calculates its factorial using a `for` loop.

## **7. Print Fibonacci Series**
Write a Python program that prints the first `N` Fibonacci numbers using a loop.

## **8. Check if a Number is Prime**
Write a Python program that checks whether a given number is prime using a loop.

## **9. Print Patterns**
Write a program to print the following pattern using nested loops:
```
*
**
***
****
*****
```

## **10. Find the Largest Number in a List**
Given a list of numbers, write a Python program that finds and prints the largest number using a loop.
